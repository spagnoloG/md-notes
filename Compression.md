# Compression
### Tar archives

- `tar -cf file.tar dir/` -archive using tar, extract `tar -xvf file.tar`
- `sudo tar -czf file.tar.gz dir/` gzip dir

### Gzip
- `gunzip archive.gz`

### Zip archives

- `unzip -x <file.zip> -d <destination_dir>` - unzip
-`zip -r {filename.zip} {foldername}` -zip


### 7zip archives
- `7z x <file.7z>` - unzip
